Data Zetu

Sahara managed the listening campaign of Data Zetu Project working with multiple partners IREX, Code For Tanzania (CfT), Tanzania Bora Initiative (TBI), SBC4D and Code For Africa (C4A).The project is aiming at building the capacity of the community and local goverment authorities to use data in making strategic decisions. Sahara collected qualitative and quantitative data from 1200 people from 16 wards in rural and urban Tanzania. The project is being implemented by funding support from MCC and PEPFAR.
